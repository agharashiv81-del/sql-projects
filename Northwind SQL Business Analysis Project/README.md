# 📊 Northwind SQL Interview Questions & Business Analytics

## 📌 Project Overview

This project contains SQL queries written on the **Northwind Database** to practice SQL for **Data Analyst interviews** and solve real-world business problems.

The project covers beginner to advanced SQL concepts, including business analysis queries that are commonly asked in interviews.
